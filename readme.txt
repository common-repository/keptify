== Description ==

Connect Keptify with your website

You can connect your keptify account with your website using this plugin. It is very simple to install and use the plugin.
This plugin is used for adding keptify code into website's footer area.

1) Upload "keptify-code" folder to the "/wp-content/plugins/" directory<br>
2) Add your userid and Submit<br>
3) Done! You have connected keptify account and your website.<br>
